# 心筋トロポニン採取実態研究：研究計画書

共同研究者による研究計画書の閲覧・レビュー用リポジトリです。

## 閲覧方法

1. [`Protocol/研究計画書.html`](Protocol/研究計画書.html) をダウンロードします。
2. ダウンロードしたファイルを Chrome、Safari、Edge などのブラウザで開きます。

HTMLは単一ファイルで、外部の画像・スタイルファイルを必要としません。ブラウザの印刷機能からPDFとして保存することもできます。

## ファイル

- `Protocol/研究計画書.md`: 編集用の正本
- `Protocol/研究計画書.html`: 共同研究者への共有・閲覧用
- `build_protocol_html.sh`: MarkdownからHTMLを再生成するスクリプト

## 更新方法

研究計画書を修正した後、プロジェクト直下で次を実行します。

```bash
bash build_protocol_html.sh
```

## 取扱い

本リポジトリには研究計画段階の未確定事項が含まれます。共同研究者以外への再配布は、研究責任者の確認後に行ってください。患者データ、認証情報、APIキーは本リポジトリに保存しないでください。
